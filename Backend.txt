-Create a Elephant SQL

 [] Invoke `psql -d <url from elephantSQL> -f starwars_postgres_create.sql`.  
 This will open the connection to your database and execute the SQL statements that will 
 create tables in your database and populate them with rows of data.  Make sure you let the script 
 run all the way through.  It will take a minute or two.
